
# Nasca Company 

1- Download Nasca Reports Export Plugin zip file
2- Go to AdminDashbored Configuration -> Plugins and click the Upload plugin or theme button.
3- A popup will appear asking you to choose a file to upload. 4- Click on the Choose File button, navigate to the downloaded zip file and select it.
5- Click on the Upload plugin or theme button, then the system will restart.
6- Find export reports plugin and click Install.

        Now you're ready to go and export your reports.
